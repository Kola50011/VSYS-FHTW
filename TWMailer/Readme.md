# TWMailer

## Coding Conventions

* CamelCase
* Variables
    * start small
* Classes
    * start with a capital letter
* Macro Names
    * CAPITAL_WITH_UNDERSCORE
    + e.g. `#define THIS_IS_A_NUMBER 1`

## Building

```bash
mkdir build
cd build
cmake ..
make all
```

## Usage

```bash
./server
./client
```
